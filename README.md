# snackbar

> Lightweight feedback notifications

## Install

```
$ npm install --save snackbar
```

## Usage

Import the snackbar.scss source file into your project.

```js
const snackbar = require('snackbar');

snackbar.show('Ohai!');
```

## API

### snackbar.show(message)

#### message 

Type `string`
